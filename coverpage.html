Project Title I-Focus
Synopsis

Throughout history, the world has been looking to design and improve user interfaces, making them more intuitive and user-friendly. It is interesting to see how we started off with the Disk Operating System (DOS) system, before advancing towards on to the common Graphic User Interface (GUI) that we use on our computers today. Yet we are never satisfied with our interfaces, but are instead taking on the challenge to constantly improve them. We now have touchscreens and motion sensing, as evident in commercial products such as our smartphones and the Xbox Kinect, but this is nonetheless insufficient.

Our project will take on interface control on a whole new level by directly connecting what the user sees on the screen with how they control the interface. I-Focus is an eye-tracking project that aims to use the latest in eye-tracking technology, combined with other forms of input, to create a set of multimodal gestures capable of efficient, effective and intuitive interface control.

I-Focus is a research-cum-application project where research with the Defence Science Organisation of Singapore (DSO) is combined with a wide range of applications that showcases how eye-tracking can be applied effectively for our target audience (the physically disabled).

We have conducted research on which input mode is most suitable for targeting tasks as well as how interfaces can be better optimised for eye-tracking in terms of selection, scrolling and navigation methods. We have created an intuitive interface in the form of a website and a bunch of games (viewable in the web report) to test and showcase our research results.
